# LAB 3 (Process Data Button Demo)

## Overview
This is a simple interactive web demo that shows how a button processes an action when clicked. The button tracks how many times it has been clicked and limits the number of actions that can be performed.

The interface provides visual feedback to the user and disables the button once the limit is reached.

---

## Features
- Tracks number of button clicks
- Displays status messages after each click
- Provides visual click feedback (button scaling)
- Disables the button after the maximum number of clicks
- Shows warning message when the limit is reached
